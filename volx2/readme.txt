---------------------------------------------------------------------------
 VOLX : AGI version 2 file extractor.
---------------------------------------------------------------------------

 Options:

   VOLX -l ???               Extract LOGIC file number ???.
   VOLX -p ???               Extract PICTURE file number ???.
   VOLX -v ???               Extract VIEW file number ???.
   VOLX -s ???               Extract SOUND file number ???.

   VOLX -o                   Extract text out of OBJECT file.
   VOLX -w                   Extract text out of WORDS.TOK file.

   VOLX -s                   Extract all LOGIC, PICTURE, VIEW and SOUND
                             resources and the OBJECT and WORDS.TOK files.

---------------------------------------------------------------------------
